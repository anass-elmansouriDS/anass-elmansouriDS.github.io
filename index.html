What i do

My aim is to become a data scientist that can help busniesses achieve operational excellence using AI technologies through extracting meaningful information from the data that they generate, Data Science is a broad field that is primarily interested in turning raw meaningless data into a pile of information that can add great value to businesses and companies. Here are some exemples of AI technologies i will use to achieve this :

Supervised Learning

Supervised learning, also known as supervised machine learning, is a subcategory of machine learning and artificial intelligence. It is defined by its use of labeled datasets to train algorithms that to classify data or predict outcomes accurately. Supervised learning can be used to solve various business problems that involve predicting a value or classification.

Unsupervised Learning

Unsupervised learning, also known as unsupervised machine learning, uses machine learning algorithms to analyze and cluster unlabeled datasets. These algorithms discover hidden patterns or data groupings without the need for human intervention. Unsupervised learning can be used in various problems that involve clustering.

Natural Language Processing

Natural language processing (NLP) is the ability of a computer program to understand human language as it is spoken and written.

Computer Vision

Computer vision is a field of artificial intelligence (AI) that enables computers and systems to derive meaningful information from digital images, videos and other visual inputs.

Deep Learning

Deep learning is a method in artificial intelligence (AI) that teaches computers to process data in a way that is inspired by the human brain. Deep learning models can recognize complex patterns in pictures, text, sounds, and other data to produce accurate insights and predictions.

Recommender Systems

A recommendation system is a class of machine learning that uses data to help predict, narrow down, and find what people are looking for among an exponentially growing number of options.
